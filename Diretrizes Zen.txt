
        Zen é uma coleção de principíos e diretrizes para escrever códigos em python de forma clara, legível e eficiente.

        Esses principios são descritos no documento chamado "The Zen of Python", este pode ser acessado digitando
        "import this" no interpretado do Python

        As diretrizes do Python são:

            1   =>  Bonito é melhor que feio
            2   =>  Explícito é melhor que implícito
            3   =>  Simples é melhor que complexo
            4   =>  Complexo é melhor que complicado
            5   =>  Plano é melhor que aninhado
            6   =>  Esparso é melhor que denso
            7   =>  Legibilidade conta
            8   =>  Casos especiais não são especiais o suficiente para quebrar as regras
            9   =>  Embora a praticidade vença a pureza
            10  =>  Erros nunca devem passar silenciosamente, a menos que explicitamente suprimidos
